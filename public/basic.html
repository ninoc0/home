Research Experience

  • Circuit Design and Modeling

    - Modeled various op-amp circuits using python to simulate their frequency responses.

    - Updating a python library called PyZero (found here) to incorporate "fitting" capabilities. This allows users to compare their data with the expected data and produce residuals plots. This work can be found here.

    - Built simple op-amp circuits to model and compare using PyZero.

    - Helped design generic PCB boards to be used for feedback loops, found here and here.

    - Modeled and built(WIP) a preliminary frequency stabilzation loop for a reference cavity.

  • Lab Experience

    Helped set up a new lab from day 1, including:

    - Testing the current and temperature controls of the laser.

    - Setting up intial polarization.

    - Creating an inital intensity stabilization loop.

    - Locking a 3 mirror ring cavity.

  • Optical Design and Modeling

    - Creating an up-to-date lab layout.

    - Creating a planned optical table design for our overall experiment.

    - Creating a planned design for our current goals, which is to set up a reference cavity.

    - Using fintrace, a library combining both finesse and g-trace, to model our reference cavity and keep an up-to-date table model.

    - Using fintrace and python to estimate the required lens to mode match our cavity.
